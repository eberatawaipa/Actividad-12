Identifica y describe las propiedades de los siguientes elementos CSS:

Propiedad Valor Inicial Descripcion
Color black Establece el color del texto.
font-family depende del navegador Define la familia tipográfica del texto.
font-size medium Establece el tamaño de la fuente.
font-weight normal Define el grosor del texto.
font-style normal Establece el estilo del texto (normal, cursiva, etc.).
font-variant normal Define la variación de la fuente, como pequeñas mayúsculas.
font depende de las propiedades Propiedad abreviada que permite establecer `font-style`, `font-variant`, `font-weight`, `font-size`, `line-height`, y `font-family`.
text-align start Define la alineación del texto dentro de su contenedor.
line-height normal Establece la altura de la línea.
text-decoration none Define la decoración del texto (subrayado, sobrelineado, etc.).
text-transform none Controla la capitalización del texto.
vertical-align baseline Define la alineación vertical de un elemento en relación con su línea base.
text-indent 0 Establece la sangría de la primera línea de texto.
letter-spacing normal Define el espacio entre caracteres.
word-spacing normal Define el espacio entre palabras.
white-space normal Controla cómo se maneja el espacio en blanco en el elemento.

Tema 4-Listas:

Propiedad Valor Inicial Descripción
list-style-type disc Define el tipo de marcador para los elementos de lista.
list-style-position outside Define la posición del marcador en relación con el contenido del ítem de la lista.
list-style-image none Define una imagen como marcador de los elementos de lista.

Propiedad Se Aplica a Valor Inicial Descripción
